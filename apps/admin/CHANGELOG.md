# 更新记录

## 0.1.0 - 2026-07-05

- StarAI 首个开源版本，提供 AI 聚合平台框架与完整前后台。
- 支持对话、图片、视频、音频、智能体工作流、开放 API、钱包推荐与卡密等核心模块。
- 后台可维护模型、渠道、角色模板、公告、用户、任务、订单、系统配置和品牌信息。
- 默认数据已按开源演示环境处理，适合二次开发、私有部署和功能验证。

## 0.1.1 - 2026-07-06

- 优化部份界面UI自适应问题。
- 新增AI漫剧功能（因SD API问题 暂未完善）。

## 0.1.2 - 2026-07-08

- 新增一键清理数据脚本（本地开发环境+线上生产环境）。
- 新增MiniMax speech 2.8 和 Music 2.6 模型 模板。
- 优化工作台UI界面。
- 后台支持工作台操作栏工具参数按钮通过 input_schema(JSON) 自定义上下摆放（默认摆放到下面，添加 `"x-placement": "top"` 就展示到上面）

## 0.2.0 - 2026-07-11

- 新增 Stripe、PayPal 与通用收银台真实在线支付，订单先进入待支付状态，仅在服务端验签回调成功后原子入账，并支持回调防重放、金额与币种校验。
- 收紧模拟支付安全边界：仅 `APP_ENV=development`、`local` 或 `test` 可即时模拟到账；`production`、`staging`、空值及未知环境全部禁止模拟支付，支付资料不完整时前台自动停用在线充值。
- 完善在线支付后台配置，包括渠道启停、币种、金额范围、回跳地址、Webhook 密钥、Stripe 密钥及 PayPal 沙箱/正式环境参数。
- 新增中、英、日、韩、越五语言支持，统一静态 UI 词典与模型、智能体、工作流、开放 API 文档等动态内容翻译链路。
- 新增后台 AI 生成缺失 UI 翻译、翻译模型测试、历史内容补翻、失败原因记录与一键重试；新建或修改内容可在后台异步翻译，不阻塞保存发布。
- 动态译文按“人工覆盖 → 目标语言译文 → 英文译文 → 中文原文”回退，语言切换直接读取已保存译文，不会在每次切换时重复调用翻译模型。
- 优化智能体工作台多语言布局，长标题、描述和标签使用保留边框的省略显示，修复卡片重叠、输入区错位及移动端进度展示问题。
- 完善部署与安全检查：本地开发使用 `.env.local`；生产部署对弱 JWT 密钥提供醒目告警但不阻断一键体验，并补充迁移校验与预发布验证流程。
